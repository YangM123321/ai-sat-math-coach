# Evaluation Plan

Build a human-labeled JSONL set covering correct work and every error category. Measure primary-category accuracy, subcategory accuracy, human-review recall, overconfidence rate, explanation groundedness, and action relevance.

Portfolio V1 engineering targets: 100% schema validity, >=90% review recall on ambiguous cases, <=5% overconfidence, and >=80% primary-category accuracy on a curated typed-work set. These are targets, not claims of proven educational effectiveness.
